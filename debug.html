BLACKHOLE Debug Console

Debug Instructions:

1. Open the main index.html file in your browser

2. Open browser developer tools (F12) and go to Console tab

3. Click the "Enter" button on the welcome screen

4. Check the console for debug messages

5. If you see "Enter button clicked!" but no auth form, there might be a CSS issue
